Core Capabilities

Everything you need for
unrestricted connectivity

Shuttle combines multiple transport protocols, intelligent routing, and adaptive performance tuning into a single, powerful toolkit.

Adaptive Congestion Control

Automatically switches between BBR and Brutal algorithms based on real-time packet loss and RTT measurements. Optimal throughput in any network condition.

Encrypted Tunneling

End-to-end encryption across all transport protocols. Your data is protected with modern cryptographic primitives including Noise IK and HMAC authentication.

Intelligent Routing

Domain trie matching, GeoIP lookups, and DNS-over-HTTPS with caching and prefetch. Define precise rules for proxy vs direct connections.

Desktop GUI

Beautiful cross-platform desktop app built with Wails + Svelte. System tray, real-time speed charts, per-app routing, and one-click connection.

Auto Transport Selection

Shuttle automatically negotiates the best available transport based on network conditions. Seamless fallback ensures you stay connected.

Mesh VPN

Connect all your devices with hub-and-spoke relay and P2P NAT traversal. STUN-based hole punching for direct device-to-device connections.
